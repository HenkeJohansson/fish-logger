# fish-logger
Fish-logger will log the position where you reported your catch, collect the weather, temperature, air pressue, wind direction, wind speed, and perhaps moon cycle.

### Technologies
* React Native
* Expo
* Firebase
* Google Maps
* SMHI or other weather API
